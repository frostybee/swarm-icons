SwarmIcons Twig Integration Demo

Native Twig support for SwarmIcons - render SVG icons with ease!

1. Basic Icon Rendering

Use the icon() function in your templates:

Home heroicons:home
User heroicons:user
Settings heroicons:cog-6-tooth
Heart lucide:heart
Star lucide:star

2. Inline Icons

Icons work great inline with text:

Navigate to Home
User Profile
Settings

3. Conditional Rendering

Check if an icon exists before rendering:

{% if icon_exists('heroicons:home') %} {{ icon('heroicons:home') }} {% else %} Icon not found {% endif %}

4. Features

  • Native Twig function support
  • Automatic HTML escaping
  • Custom attributes via array syntax
  • Accessibility (ARIA) built-in
  • Silent mode for missing icons

5. Multiple Icon Sets

Access icons from different providers:

Heroicons
Lucide

6. Accessibility

Icons with aria-label get role="img" automatically:

Labeled icon (has role="img")
Decorative icon (has aria-hidden="true")
