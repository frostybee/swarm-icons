## Setup

```bash
composer install
```

## Available Commands

| Command | Description |
|---|---|
| `composer test` | Run PHPUnit tests |
| `composer phpstan` | Run PHPStan static analysis (level 8) |
| `composer cs-check` | Check code style (PER-CS2.0, dry-run) |
| `composer cs-fix` | Auto-fix code style |
| `composer test-all` | PHPStan + CS check + tests (sequential) |

## Running Individual Tests

```bash
# Run a single test file
vendor/bin/phpunit tests/Unit/IconTest.php

# Run a single test method
vendor/bin/phpunit --filter test_method_name

# Run all tests in a directory
vendor/bin/phpunit tests/Unit/
```

## CI Pipeline

GitHub Actions runs on push/PR to `main`:

- **Tests**: PHP 8.2, 8.3, 8.4, 8.5 matrix
- **PHPStan**: PHP 8.4
- **CS check**: PHP 8.4

## Code Style

The project uses **PER-CS2.0** with `PHP82Migration` rules. Key conventions:

- `declare(strict_types=1)` in every PHP file
- Alpha-sorted `use` imports
- 4-space indentation for PHP/JSON, 2-space for YAML/HTML/SVG
- LF line endings
- Test method names: `test_snake_case_description()` extending `PHPUnit\Framework\TestCase`

Run `composer cs-fix` to auto-format before committing.

## CLI Tool (Development)

```bash
# List available CLI commands
php bin/swarm-icons list

# Download icon sets for testing
php bin/swarm-icons json:download mdi tabler

# Search for icons
php bin/swarm-icons icon:search tabler arrow

# Clear the icon cache
php bin/swarm-icons cache:clear
```
